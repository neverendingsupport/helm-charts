# nes-node-web

`nes-node-web` is a smaller chart for Node-based web services.

Use it when:

- the workload already matches the chart shape closely
- you do not need the broader integration surface of `universal-chart`

For most new HeroDevs services, `universal-chart` is still the default. Reach
for `nes-node-web` only when it is a better fit than the general-purpose chart.

The full values reference is generated by `helm-docs` into the chart
`README.md` during pre-commit and is included here in Backstage as the local
[Generated Reference](reference.md) page.
